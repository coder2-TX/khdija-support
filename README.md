# Khdija Support Messages Package

حزمة Laravel لنظام مراسلة الدعم بين المنشآت والإدارة.

## المميزات

- ✅ نظام مراسلة متعدد المستخدمين
- ✅ دعم الصلاحيات (سوبر أدمن، منشآت)
- ✅ عداد الرسائل غير المقروءة
- ✅ تحديث تلقائي للدردشة
- ✅ دعم RTL/LTR
- ✅ واجهة مستخدم متجاوبة
- ✅ تكوين مرن

## التثبيت

```bash
composer require khdija/support-messages
